(módulo de magisk)(https://github.com/evdenis/disk)

(Tener root)

#Entrar como superusuario (Si no funciona escribe pkg install tsu)
Tsu 

#Hacer un archivo con la tabla de particiones
sfdisk -d /dev/block/sda > /sdcard/sda.part

#Salir del modo superusuario
Exit

#Instalar nano
pkg install nano

#Entrar como superusuario
Tsu

#Abrir el archivo creado con nano
nano /sdcard/sda.part

(Borrar particiones necesarias (4-7))

#Importar la nueva tabla de particionds
sfdisk --no-reread -f /dev/block/sda < /sdcard/sda.part
